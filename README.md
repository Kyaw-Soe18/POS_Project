Point of Sale (POS) System
Overview

This is a modern Point of Sale (POS) system built with Laravel. It allows businesses to efficiently manage products, orders, payments, and customers. The system also supports social login via Google and GitHub, making user authentication quick and easy. The dashboard provides an overview of sales and analytics, and the application is fully responsive for desktop and mobile devices.

Tech Stack

Backend: Laravel (PHP)

Frontend: Blade Templates, HTML, CSS, JavaScript

Database: MySQL

Authentication: Laravel Breeze / Socialite (Google & GitHub login)

Other Tools: Composer, NPM, Vite

Features

User authentication with email/password

Social login with Google and GitHub

Product management (add, edit, delete)

Order creation and sales tracking

Payment processing

Customer management

Dashboard with sales statistics and analytics

Responsive design for desktop and mobile
